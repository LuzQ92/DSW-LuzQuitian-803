Tema Seleccionado: Selectores CSS

Los selectores CSS son una parte fundamental de la sintaxis de CSS. Permiten seleccionar y estilizar elementos específicos en una página web. Aquí te mostraremos algunos ejemplos de selectores comunes:

Selector de elemento

El selector de elemento selecciona todos los elementos de un tipo específico. Por ejemplo:

p {
            color: blue;
          }

Selector de clase

El selector de clase selecciona elementos que tienen un atributo de clase específico. Por ejemplo: 

.destacado {
            font-weight: bold;
          }

Selector de ID

El selector de ID selecciona un único elemento con un atributo de ID específico. Por ejemplo:

#logo {
            width: 200px;
          }

Selector de atributo

El selector de atributo selecciona elementos que tienen un atributo específico. Por ejemplo:

input[type="text"] {
            border: 1px solid gray;
          }

Selector descendente

El selector descendente selecciona elementos secundarios dentro de un elemento principal. Por ejemplo:

ul li {
            list-style-type: circle;
          }

Selector de hermanos adyacentes

El selector de hermanos adyacentes selecciona el siguiente elemento inmediato que sigue al elemento actual. Por ejemplo:

h2 + p {
            color: red;
          }

Selector de pseudo-clase

El selector de pseudo-clase selecciona elementos en función de un estado o acción específica. Por ejemplo:

a:hover {
            text-decoration: underline;
          }

Selector de pseudo-elemento

El selector de pseudo-elemento selecciona partes específicas de un elemento. Por ejemplo:

p::first-line {
            font-weight: bold;
          }

Combinación de selectores

También es posible combinar varios selectores para seleccionar elementos de forma más precisa. Por ejemplo:

h1, h2 {
            color: green;
          }
